Functions

make_class (base, mt)
Create a new class prototype or instance.

Parameters:

  • base Table used as a base class. Creates a new empty table if omitted.
  • mt Metatable for the new class. Sets base as metatable if omitted. The __index metamethod of this table will be set to the base argument.

Returns:

    New class table.

Creates a callable table.

Parameters:

Returns:

    Table with the __call metamethod set to the provided function.

map_copy (dest, src, fn)
Copies key-value pairs from one table to another and applies a function to the values.

Parameters:

  • dest Destination table.
  • src Source table.
  • fn Function applied to values before assigning them. It's called as fn(value, key) for each key in src, then the result is assigned to dest[key], unless it's nil.

Returns:

    The dest argument.
lazy_loader (fn)
Creates a lazy table that loads its contents on field access.

Parameters:

  • fn Function that returns the table content. This function will be called on the first field read attempt. The returned table fields will be copied to this table.

Returns:

    New lazy table.
make_set (...)
Creates a set table from the keys in the arguments.

Parameters:

  • ... Keys of the table. The values will be set to true.

Returns:

    New set table.
check_args (args, rules, res_name)
Checks if the arguments in the specified table match the rules.

Parameters:

  • args Table with arguments to be checked.
  • rules Rules to check against.
  • res_name Name of the resource that is being checked (for error messages)

Returns:

  1. true if args is valid, otherwise false.
  2. The error string if args is invalid.
shift_pcall_error (ok, ...)
Removes the first value from a pcall result and returns errors in Lua style.

Parameters:

  • ok Success status from pcall.
  • ... List of returned values.

Returns:

  1. On success, the function return values. On failure nil.
  2. The error string.

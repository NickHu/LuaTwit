Functions

new (base, mt)
Create a new class prototype or instance.

Parameters:

  • base Table used as a base class. Creates a new empty table if omitted.
  • mt Metatable for the new class. Sets base as metatable if omitted. The __index metamethod of this table will be set to the base argument.

Returns:

    New class table.
bless (tbl, mt)
Converts a table into a class instance of the supplied type.

Parameters:

  • tbl Table to be converted. If it's not a table it will be returned unmodified.
  • mt Metatable that defines the type of the object.

Returns:

    The table argument with type as it's metatable.
assertx (cond, message, level)
Assert implementation with an extra 'level' argument.

Parameters:

  • cond Condition that must be true. If false it triggers an error.
  • message Error message to display when cond is false.
  • level Call stack level that the error function uses to display the error source.
load_file (filename, env)
Loads a Lua chunk from a file and executes it on it's own environment.

Parameters:

  • filename Lua source file.
  • env Environment used to run the code. Creates a new empty table if omitted.

Returns:

  1. The environment where the code was executed.
  2. true if no errors found when executing the code.
  3. Value(s) returned by the Lua chunk, or the error message.

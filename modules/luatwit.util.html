Functions

new (base, mt)
Create a new class prototype or instance.

Parameters:

  • base Table used as a base class. Creates a new empty table if omitted.
  • mt Metatable for the new class. Sets base as metatable if omitted. The __index metamethod of this table will be set to the base argument.

Returns:

    New class table.
load_file (filename, env)
Loads a Lua chunk from a file and executes it on it's own environment.

Parameters:

  • filename Lua source file.
  • env Environment used to run the code. Creates a new empty table if omitted.

Returns:

  1. The environment where the code was executed.
  2. true if no errors found when executing the code.
  3. Value(s) returned by the Lua chunk, or the error message.

Inherits methamethods from one table to another by copying them.

map_copy (dest, src, fn)
Copies key-value pairs from one table to another and applies a function to the values.

Parameters:

  • dest Destination table.
  • src Source table.
  • fn Function applied to values before assigning them. It's called as fn(value, key) for each key in src, then the result is assigned to dest[key], unless it's nil.

Returns:

    The dest argument.
lazy_loader (fn)
Creates a lazy table that loads its contents on field access.

Parameters:

  • fn Function that returns the table content. This function will be called on the first field read attempt. The returned table fields will be copied to this table.

Returns:

    New lazy table.
set (...)
Creates a set table from the keys in the arguments.

Parameters:

  • ... Keys of the table. The values will be set to true.

Returns:

    New set table.

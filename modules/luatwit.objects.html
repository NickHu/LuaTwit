access_token:save (filename)
Saves the content of an access_token object into a file. The output file can be loaded with luatwit.load_keys .

Parameters:

  • filename Name of the destination file.

Returns:

    The access_token itself.

Fields

access_token
Access token returned by luatwit.api:confirm_login . It's the result of the user authorizing the app, and contains the keys necessary to make API calls.
